COURSES

Kali Linux is an open source Linux distribution for advanced Penetration Testing or Security Auditing.

Metasploit is a powerful tool used by both ethical hackers and cyber-criminals. It helps find vulnerabilities on networks and servers.

Burp Suite is a powerful penetration testing vulnerability finding tool. It is often used for checking web application security.

Wireshark is the go-to network protocol analysis tool. Anyone from government agencies to schools use wireshark for troubleshooting and teaching purposes.
